#  Daily Water Intake Reminder

A smart and user-friendly web application that helps users track their daily water intake, set reminders, and build healthy hydration habits.

---

##  Features

-  Daily water intake reminder
-  Smart water tracking
-  Progress view (daily hydration status)
-  Healthy habit support
-  Responsive design (mobile & desktop friendly)

---

## How It Works

1. User sets their daily water intake goal  
2. The app reminds the user to drink water at regular intervals  
3. User logs water intake  
4. Progress is visually displayed to motivate healthy habits
